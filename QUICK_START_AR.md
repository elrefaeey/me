# 🚀 البداية السريعة - Quick Start

## خطوات الإعداد (5 دقائق)

### 1. إعداد Firebase Authentication
```
1. افتح: https://console.firebase.google.com/
2. اختر مشروع: elrefaey-db24e
3. Authentication → Get Started
4. Sign-in method → Email/Password → Enable
5. Users → Add user
   - Email: admin@example.com (أو أي بريد تريده)
   - Password: كلمة مرور قوية
```

### 2. إعداد Firestore Database
```
1. Firestore Database → Create database
2. Start in production mode
3. اختر الموقع → Enable
4. Rules → انسخ من ملف firebase-rules/firestore.rules
```

### 3. إعداد Storage
```
1. Storage → Get started
2. Start in production mode → Done
3. Rules → انسخ من ملف firebase-rules/storage.rules
```

### 4. تشغيل المشروع
```bash
npm install
npm run dev
```

### 5. الدخول للوحة التحكم
```
1. افتح: http://localhost:5173
2. اضغط على 🔒 في الزاوية السفلية اليمنى
3. سجل دخول بالبريد وكلمة المرور
4. ابدأ التعديل!
```
