## 🚀 البداية السريعة

### 1. تثبيت المكتبات
```bash
npm install
```

### 2. إعداد Firebase
اتبع التعليمات في: `QUICK_START_AR.md`

الخطوات الأساسية:
1. فعّل Authentication (Email/Password)
2. أنشئ حساب Admin
3. فعّل Firestore Database
4. فعّل Storage
5. انسخ قواعد الأمان من مجلد `firebase-rules/`

### 3. تشغيل المشروع
```bash
npm run dev
```

### 4. الدخول للوحة التحكم
1. افتح الموقع
2. اضغط على 🔒 في الزاوية السفلية اليمنى
3. سجل دخول
4. ابدأ التعديل!

## 🔐 الأمان

- ✅ Firebase Authentication للمصادقة
- ✅ Protected Routes للصفحات المحمية
- ✅ Firestore Rules: القراءة للجميع، الكتابة للأدمن فقط
- ✅ Storage Rules: القراءة للجميع، الكتابة للأدمن فقط
- ✅ زر القفل مخفي ولا يظهر بشكل واضح

## 🛠️ التقنيات المستخدمة

- **Frontend**: React + TypeScript + Vite
- **UI**: Shadcn/ui + Tailwind CSS
- **Backend**: Firebase
  - Authentication (Email/Password)
  - Firestore Database
  - Storage
- **Routing**: React Router v6
- **State Management**: React Context API

## 🎨 المميزات التقنية

### رفع الصور
- دعم رفع الصور مباشرة من الجهاز
- دعم إضافة روابط صور خارجية
- حفظ الصور في Firebase Storage
- معاينة الصور قبل الحفظ

### إدارة المشاريع
- CRUD كامل (Create, Read, Update, Delete)
- ترتيب المشاريع حسب الأولوية
- إخفاء/إظهار بدون حذف
- روابط متعددة (Demo + GitHub)

### وسائل التواصل
- إدارة 5 منصات
- إخفاء/إظهار كل منصة
- الأيقونات تظهر فقط إذا كان الرابط موجود
- تحديث فوري

## 🔄 سير العمل

1. **الزائر** يفتح الموقع → يرى المحتوى فقط
2. **الأدمن** يضغط على 🔒 → يسجل دخول
3. **الأدمن** يعدل المحتوى → يحفظ
4. **التغييرات** تظهر مباشرة في الموقع
5. **الزوار** يرون المحتوى المحدث
