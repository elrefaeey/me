## المرحلة 1: الإعداد الأولي (10 دقائق)

### الخطوة 1: إعداد Firebase
```
⏱️ الوقت: 5 دقائق

1. افتح متصفحك واذهب إلى:
   https://console.firebase.google.com/

2. اختر مشروعك: elrefaey-db24e

3. من القائمة الجانبية، اضغط على "Authentication"

4. اضغط "Get Started"

5. اختر "Email/Password" من قائمة Sign-in methods

6. فعّل الخيار الأول (Email/Password)

7. اضغط "Save"

8. اذهب إلى تبويب "Users"

9. اضغط "Add user"

10. أدخل:
    - Email: admin@yoursite.com
    - Password: كلمة مرور قوية (مثل: Admin@123456)

11. اضغط "Add user"

✅ تم! الآن لديك حساب Admin
```

### الخطوة 2: إعداد Firestore
```
⏱️ الوقت: 2 دقيقة

1. من القائمة الجانبية، اضغط "Firestore Database"

2. اضغط "Create database"

3. اختر "Start in production mode"

4. اختر الموقع الأقرب لك (مثل: europe-west)

5. اضغط "Enable"

6. انتظر حتى يتم الإنشاء

7. اذهب إلى تبويب "Rules"

8. احذف القواعد الموجودة والصق هذا:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read: if true;
    }
    match /content/{document} {
      allow write: if request.auth != null;
    }
    match /projects/{document} {
      allow write: if request.auth != null;
    }
  }
}

9. اضغط "Publish"

✅ تم! Firestore جاهز
```

### الخطوة 3: إعداد Storage
```
⏱️ الوقت: 2 دقيقة

1. من القائمة الجانبية، اضغط "Storage"

2. اضغط "Get started"

3. اختر "Start in production mode"

4. اضغط "Next"

5. اضغط "Done"

6. اذهب إلى تبويب "Rules"

7. احذف القواعد الموجودة والصق هذا:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}

8. اضغط "Publish"

✅ تم! Storage جاهز
```

### الخطوة 4: تشغيل المشروع
```
⏱️ الوقت: 1 دقيقة

1. افتح Terminal في مجلد المشروع

2. اكتب:
   npm install

3. انتظر حتى تنتهي التثبيت

4. اكتب:
   npm run dev

5. افتح المتصفح على:
   http://localhost:5173

✅ تم! المشروع يعمل
```

## المرحلة 2: تسجيل الدخول (1 دقيقة)

```

4. اضغط عليها

7. اضغط "تسجيل الدخول"
```

## المرحلة 3: تعديل Hero Section (3 دقائق)

```
1. في لوحة التحكم، اضغط على تبويب "Hero Section"

2. ستجد 4 حقول:

   أ. الاسم:
      - أدخل اسمك الكامل
      - مثال: أحمد الرفاعي

   ب. العنوان:
      - أدخل مسماك الوظيفي
      - مثال: Full Stack Developer

   ج. الوصف:
      - أدخل جملة قصيرة عنك
      - مثال: Building amazing web experiences

   د. الصورة:
      - الطريقة 1: أدخل رابط صورة
        مثال: https://i.imgur.com/yourimage.jpg
      
      - الطريقة 2: ارفع صورة من جهازك
        اضغط "Choose File" واختر الصورة
        انتظر حتى يتم الرفع
        ستظهر رسالة "تم رفع الصورة بنجاح"

3. بعد إدخال جميع البيانات، اضغط "حفظ التغييرات"

4. ستظهر رسالة "تم حفظ التغييرات بنجاح"

5. اذهب إلى الصفحة الرئيسية لرؤية التغييرات

✅ تم! Hero Section محدث
```

## المرحلة 4: تعديل About Section (2 دقيقة)

```
1. اضغط على تبويب "About"

2. ستجد مربع نص كبير

3. اكتب نبذة عنك:
   - من أنت؟
   - ماذا تفعل؟
   - ما هي خبراتك؟
   - ما هي اهتماماتك؟

4. مثال:
   "مطور ويب متخصص في بناء تطبيقات الويب الحديثة
   باستخدام React و Node.js. لدي خبرة 5 سنوات في
   تطوير واجهات المستخدم التفاعلية وبناء APIs قوية."

5. اضغط "حفظ التغييرات"

6. ستظهر رسالة "تم حفظ التغييرات بنجاح"

✅ تم! About Section محدث
```

## المرحلة 5: إضافة مشروع (5 دقائق)

```
1. اضغط على تبويب "Projects"

2. اضغط زر "إضافة مشروع" (الزر الأخضر في الأعلى)

3. ستفتح نافذة منبثقة، املأ الحقول:

   أ. عنوان المشروع:
      مثال: متجر إلكتروني متكامل

   ب. الوصف:
      مثال: منصة تجارة إلكترونية كاملة مع نظام دفع آمن

   ج. رابط الصورة:
      - أدخل رابط صورة، أو
      - ارفع صورة من جهازك

   د. رابط المشروع:
      مثال: https://demo-ecommerce.com

   هـ. رابط GitHub:
      مثال: https://github.com/username/ecommerce

   و. ترتيب العرض:
      - 1 للمشروع الأول
      - 2 للمشروع الثاني
      - وهكذا...

   ز. مرئي للزوار:
      - فعّل الزر إذا كنت تريد إظهار المشروع
      - أطفئه إذا كنت تريد إخفاءه

4. اضغط "حفظ"

5. ستظهر رسالة "تم إضافة المشروع بنجاح"

6. سيظهر المشروع في القائمة

7. كرر العملية لإضافة مشاريع أخرى

✅ تم! تم إضافة المشروع
```

## المرحلة 6: تعديل مشروع موجود (2 دقيقة)

```
1. في قائمة المشاريع، ابحث عن المشروع

2. اضغط زر "تعديل" بجانب المشروع

3. ستفتح نافذة التعديل

4. عدّل ما تريد

5. اضغط "حفظ"

6. ستظهر رسالة "تم تحديث المشروع بنجاح"

✅ تم! تم تعديل المشروع
```

## المرحلة 7: حذف مشروع (30 ثانية)

```
1. في قائمة المشاريع، ابحث عن المشروع

2. اضغط زر "حذف" (الزر الأحمر)

3. ستظهر رسالة تأكيد

4. اضغط "OK" للتأكيد

5. ستظهر رسالة "تم حذف المشروع بنجاح"

⚠️ تحذير: الحذف نهائي ولا يمكن التراجع عنه!

✅ تم! تم حذف المشروع
```

---

## المرحلة 8: إخفاء/إظهار مشروع (10 ثواني)

```
1. في قائمة المشاريع، ابحث عن المشروع

2. اضغط زر "مرئي" أو "مخفي"

3. سيتغير الزر تلقائيًا

4. ستظهر رسالة "تم تحديث حالة العرض"

5. المشاريع المخفية لا تظهر للزوار

💡 فائدة: يمكنك إخفاء المشروع بدون حذفه!

✅ تم! تم تغيير حالة العرض
```

## المرحلة 9: إدارة وسائل التواصل (5 دقائق)

```
1. اضغط على تبويب "Contact"

2. املأ معلومات التواصل:
   - الاسم
   - العنوان الفرعي
   - الصورة (رابط أو رفع)

3. لكل منصة تواصل:

   أ. Instagram:
      - أدخل الرابط: https://instagram.com/username
      - فعّل "إظهار في الموقع" إذا كنت تريد إظهاره

   ب. WhatsApp:
      - أدخل الرابط: https://wa.me/201234567890
      - فعّل "إظهار في الموقع"

   ج. Facebook:
      - أدخل الرابط: https://facebook.com/username
      - فعّل "إظهار في الموقع"

   د. LinkedIn:
      - أدخل الرابط: https://linkedin.com/in/username
      - فعّل "إظهار في الموقع"

   هـ. GitHub:
      - أدخل الرابط: https://github.com/username
      - فعّل "إظهار في الموقع"

4. اضغط "حفظ جميع التغييرات"

5. ستظهر رسالة "تم حفظ التغييرات بنجاح"

💡 نصيحة: إذا تركت الرابط فارغًا، لن تظهر الأيقونة

✅ تم! وسائل التواصل محدثة
```

## 💡 نصائح مهمة

### عند رفع الصور:
- ✅ استخدم صور بجودة عالية
- ✅ حجم الصورة أقل من 5MB
- ✅ صيغ مدعومة: JPG, PNG, WebP
- ❌ لا ترفع صور كبيرة جدًا

### عند إدخال الروابط:
- ✅ تأكد من صحة الرابط
- ✅ استخدم HTTPS
- ✅ اختبر الرابط في المتصفح
- ❌ لا تنسى حفظ التغييرات

### عند إضافة المشاريع:
- ✅ رتب المشاريع حسب الأهمية
- ✅ اكتب وصف واضح
- ✅ أضف روابط Demo و GitHub
- ❌ لا تترك حقول فارغة

### عند إدارة وسائل التواصل:
- ✅ أضف فقط الحسابات النشطة
- ✅ تأكد من صحة الروابط
- ✅ يمكنك إخفاء أي منصة
- ❌ لا تنسى حفظ التغييرات

---

## ⚠️ تحذيرات

1. **احفظ دائمًا قبل الانتقال**
   - اضغط "حفظ التغييرات" قبل الانتقال لقسم آخر
   - وإلا ستفقد التغييرات!

2. **الحذف نهائي**
   - حذف المشروع لا يمكن التراجع عنه
   - استخدم "إخفاء" بدلاً من "حذف" إذا كنت غير متأكد

3. **حجم الصور**
   - لا ترفع صور كبيرة جدًا
   - قد يفشل الرفع إذا كانت الصورة أكبر من 5MB

4. **الروابط**
   - تأكد من صحة الروابط
   - الروابط الخاطئة لن تعمل للزوار
