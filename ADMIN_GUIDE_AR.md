# 🔒 دليل لوحة التحكم - Admin Dashboard

## 🚀 البداية السريعة

### الخطوة 1: إعداد Firebase
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/)
2. افتح مشروعك: `elrefaey-db24e`
3. فعّل **Authentication** (Email/Password)
4. أنشئ حساب Admin من تبويب Users
5. فعّل **Firestore Database**
6. فعّل **Storage**

### الخطوة 2: تشغيل المشروع
```bash
npm install
npm run dev
```

### الخطوة 3: الدخول للوحة التحكم
1. افتح الموقع
2. اضغط على أيقونة القفل 🔒 في الزاوية السفلية اليمنى
3. سجل دخول بحساب الأدمن
4. ابدأ التعديل!

## 💡 نصائح مهمة

### رفع الصور
- يمكنك رفع صورة مباشرة من جهازك
- أو إضافة رابط صورة من الإنترنت
- الصور المرفوعة تُحفظ في Firebase Storage

### الحفظ
- اضغط "حفظ التغييرات" بعد كل تعديل
- التغييرات تظهر مباشرة في الموقع
- لا تنسى الحفظ قبل الانتقال لقسم آخر!

### الأمان
- فقط الحسابات المسجلة في Firebase يمكنها الدخول
- الزوار يمكنهم رؤية المحتوى فقط
- لا يمكن للزوار التعديل

### وسائل التواصل
- إذا تركت الرابط فارغًا، لن تظهر الأيقونة
- يمكنك إخفاء أي منصة بدون حذف الرابط
- الروابط المخفية لا تظهر للزوار

---

## 🔧 حل المشاكل

### لا أستطيع تسجيل الدخول
- تأكد من إنشاء حساب في Firebase Authentication
- تأكد من تفعيل Email/Password في Sign-in methods
- تحقق من البريد الإلكتروني وكلمة المرور

### الصور لا تُرفع
- تأكد من تفعيل Firebase Storage
- تحقق من قواعد الأمان (Security Rules)
- حجم الصورة يجب أن يكون معقول (أقل من 5MB)

### التغييرات لا تظهر
- تأكد من الضغط على "حفظ التغييرات"
- حدّث الصفحة الرئيسية (F5)
- تحقق من اتصال الإنترنت
